# Análisis OSINT pasivo de INDRA

Repositorio del laboratorio de reconocimiento pasivo sobre la empresa INDRA, seleccionada del IBEX35.

## Objetivo

Realizar una investigación OSINT pasiva con enfoque académico, documentando hallazgos, metodología, resultados y conclusiones.

## Estructura del repositorio

- `docs/` → informe en Typst, bibliografía e imágenes
- `data/raw/` → datos en bruto obtenidos durante la investigación
- `data/processed/` → datos procesados y tablas finales
- `P1_Reconocimiento_pasivo/src/` → scripts y herramientas auxiliares
- `P1_Reconocimiento_pasivo/results/` → resultados técnicos del reconocimiento
